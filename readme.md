# smart-home

## backend

First, run :
```sh
cd api
npm i
```

run server:
```sh
npm run start
```

test API in [swagger](http://localhost:3000/api-docs)


## frontend

First, run:
```sh
cd template
npm i
```

run frontend:
```sh
npm run start
```

[login page](http://localhost:5173/#/auth/login)
